<h1 id="trpg">TRPG</h1>

此處包含了 TRPG (Tabletop Role Playing Game) 會使用到的常見擲骰功能。

目前已支援 CoC 與 NC 骰子囉。（但是我沒玩過 DX 所以就不會寫...）


<h2 id="coc">CoC</h2>

CoC 為 Call of Cthulhu 的簡寫，相關說明請參考網路資源（作者好懶），打個指令可以看說明：

    cc 說明
    
    CoC 擲骰範例: 
     → ccb<=   六版擲骰 
     → cc<=    七版擲骰 
     → cc(N)<= 獎勵骰 [-2~2] 
     → cc>     幕間成長 
     → 2d4+1 / 2D10+1d2  直接擲骰 
     → 7 3d6 / 5 2d6+6   多筆輸出 

    CoC 一鍵創角: 
     → cc 核心創角 [年齡]  以核心規則創角（含年齡調整）
     → cc 悠子創角  主要屬性骰七取五，次要屬性骰四取三，LUK骰二取一
    


<h3 id="coc6">ccb<=六版擲骰</h3>
    
CoC 六版擲骰語法，語法為，問題可為空白：
    
    ccb<=數字 (問題)
    
比方說：

    ccb<=50 San Check
    (1D100<=50) → 48 → 通常成功
    
    
<h3 id="coc7">cc<=七版擲骰</h3>

CoC 七版擲骰語法，語法為，問題可為空白：

    cc<=數字 (問題)

比方說：

    cc<=50 SanCheck
    (1D100<=50) → 64 → 失敗
    
都更新了丟出來的數字還是一樣...
    
    
<h3 id="award">cc(N)<=獎勵骰</h3>

語法為，問題同樣可為空白：

    cc(N)<=數字 (問題)
    
比方說：

    cc(2)<=50 偵查
    (1D100<=50) → 80、50、90 → 50 → 通常成功
    
    
<h3 id="growup"> cc> 幕間成長</h3>

幕間成長是 0.5.0 版之後新增的功能，感謝 [retsnimle][] 分享。
這裡的用法為：

    cc>數字 (內容)
    
比方說：

    cc>50 偵查
    (1D100>50) → 90 → 成功成長10點
    

    
<h3 id="roll">直接丟骰子</h3>

~~直接輸入 roll 會跳出使用說明，像是：~~
0.5.0 版本之後，支援直接的四則運算了，直接輸入骰子數量即可，比方說：

    2D10
    基本擲骰：(8+2) = 10
    
    3D6+1D6
    基本擲骰：(2+2+1)+(6) = 11
    


<h3 id="multidice">多筆輸出</h3>
     
要多筆輸出就是先打你要的次數，再空一格打骰數，比方說

    7 3d6
    
    複數擲骰：
    1# (1+3+3) = 7
    2# (1+3+4) = 8
    3# (2+2+3) = 7
    4# (1+5+6) = 12
    5# (5+2+1) = 8
    6# (1+2+3) = 6
    7# (6+3+3) = 12
     

<h3 id="quickrolecreate">一鍵創角</h3>

在 0.6.0 版本內，提供了輸入指令快速創角的功能，有兩種方式，現列如下：


以核心規則創角（含年齡調整）的指令為：

    cc 核心創角 [年齡] 
    
舉個例子：

    cc 核心創角 39
    
    調查員年齡設為：39
    ==
    年齡調整：可做1次EDU的成長擲骰。
    ==

    ＳＴＲ：(2+5+1)*5 = 40
    ＣＯＮ：(6+3+6)*5 = 75
    ＤＥＸ：(4+1+5)*5 = 50
    ＡＰＰ：(4+3+2)*5 = 45
    ＰＯＷ：(3+3+2)*5 = 40
    ＳＩＺ：((1+5)+6)*5 = 60
    ＩＮＴ：((2+1)+6)*5 = 45
    ==
    ＥＤＵ初始值：((1+2)+6)*5 = 45
    第1次EDU成長 → 72 → 成功成長8點

    ＥＤＵ最終值：53
    ==

    LUK：(1+2+2)*5 = 25


另外還有一種，以悠子房規為主的創角功能，主要屬性骰七取五，次要屬性骰四取三，LUK 骰二取一，指令為：

    cc 悠子創角 
    
同樣舉個例子：

    cc 悠子創角
    
    骰七次3D6取五次，
    決定STR、CON、DEX、APP、POW。

    1# (3+6+3)*5 = 60
    2# (5+3+6)*5 = 70
    3# (3+2+1)*5 = 30
    4# (6+3+3)*5 = 60
    5# (1+6+2)*5 = 45
    6# (4+2+3)*5 = 45
    7# (3+4+2)*5 = 45
    ==
    骰四次2D6+6取三次，
    決定SIZ、INT、EDU。

    1# ((5+3)+6)*5 = 70
    2# ((5+1)+6)*5 = 60
    3# ((1+3)+6)*5 = 50
    4# ((1+1)+6)*5 = 40
    ==
    骰兩次3D6取一次，
    決定LUK。

    1# (6+3+2)*5 = 55
    2# (6+5+6)*5 = 85 

這數值好爛 XD

<h3 id="ncdice">NC 擲骰介紹</h3>

此版本已提供 NC 常見的擲骰功能囉（格式則參考跑團平台──凍豆腐）。
如要擲骰的話請參照下列格式：

    (骰數)NC/NA (問題)
    
比方說：

    1NC
    7[7] → 成功
    
    4nc-2
    5[2,1,5,0] → 大失敗
    
    1NA+2 反物資
    6[6] → 成功 ; 反物資
    
已支援同時大小寫判斷。

<h2 id="tarot">塔羅牌</h2>

此程式提供塔羅牌的抽牌功能，如果直接輸入 tarot 的話會跳出說明：
    
    tarot
    
    每日運勢: 22 張，請用 tarot daily/每日 
    時間之流: 79 張，請用 tarot time/時間 (問題) 
    大十字: 79 張，請用 tarot cross/大十字 (問題) 
    儘管玩玩看
    
也可以抽每日運勢，會從 22 張大牌內抽一張出來：

    tarot daily (問題)
    
    命運之輪 ＋
    
如果後面不加關鍵字，單純問問題，則會從 79 張牌（嘿嘿）內抽一張出來：

    tarot 隨便抽
    
    權杖一 ＋
    

<h3 id="daily">每日運勢 / 每日一抽</h3>

每天來一張吧！相關語法為：

    tarot daily/每日 (問題)
    
比方說這樣輸入都可以：

    tarot daily 
    魔術師 ＋
    
    tarot 每日 隨便問問題
    命運之輪 － ; 隨便問問題


<h3 id="time">牌陣：時間之流</h3>

這是塔羅一個很經典的牌陣：時間之流，可以在精簡的分析內看到過去、現在與未來的概觀。如果想要用此牌陣的話，相關語法為：

    tarot time/時間 (問題)
    
比方說：

    tarot time 還是隨便問
    
    還是隨便問: 
    過去: 惡魔 ＋
    現在: 錢幣八 ＋
    未來: 隱者 －


<h3 id="cross">牌陣：塞爾特大十字</h3>

這是塔羅一個非常實用的牌陣，此牌陣能夠清楚地從各個面相顯示出問題的核心，真的遇到問題就用這個吧，甚至不需要學其他牌陣（大誤），相關語法如下：

    tarot 大十字/cross (問題)

    tarot cross 作者帥不帥

    現況: 權杖皇后 ＋

    未來: 寶劍騎士 －

<h2 id="baobei">擲筊</h2>

此杯杯機器人並提供了簡單的擲筊功能，設計含意如下：

* 使用 0 為陰數， 1 為陽數，用陰陽交替的方式來取杯。
* 為了呈現出擲筊的多樣性，因此多引入數字 2 當做輔助數。
* 程式會自動丟三次杯，如特定杯有超過一定次數則取杯，反之則沒杯。
* 此程式會出現聖杯、笑杯、蓋杯、沒杯、立杯等現象。
* 沒杯的可能性很高。


另外在這邊簡單定義一下各種杯的含意：

* 聖杯：好、贊成、OK。（不是聖杯戰爭那個。）
* 笑杯：答案自在心中不需要問、神明在笑、問的很接近了但又差一點點。
* 蓋杯：不好、反對、NO。
* 沒杯：不想回答、不置可否、問的問題太過偏遠、不切實際。
* 立杯：哈哈神祕現象哈哈，請自行解讀。

如果出現了三聖/笑/蓋杯，那就表示所得結果很穩定呢！


<h3 id="cup">來丟個杯吧</h3>

如果直接輸入以下語法，則會跳出說明：

    cup 說明
    
    常見的擲筊 
    請用 丟杯/擲筊/cup (問題) 
    會出現聖杯、笑杯、蓋杯、沒杯與立杯等結果
    
    
如果想要丟杯，則使用下列語法：

    cup (問題)
    
比方說：

    cup
    聖杯
    
    cup 週末洗衣服好不好
    沒杯 ; 週末洗衣服好不好
    
代表這種小問題不需要來問我 T_T
