TarotLineBot
==================

這是一個 LINE 專用的自動占卜 & TRPG 擲骰機器人，支援如下功能：

*   [TRPG](#trpg)
    *   [COC](#coc)
        *   [ccb   六版擲骰](#coc6)
        *   [cc    七版擲骰](#coc7)
        *   [cc(N) 獎勵骰](#award)
        *   [roll  直接丟骰子](#roll)
        *   [支援快速產生六版](#createcoc6) / [七版角色卡數值](#createcoc7)
    *   [NC](#nechronica)
        *   [1NC/4NC+2 擲骰介紹](#ncdice)
*   [占卜](#divine)
    *   [塔羅牌](#tarot)
        *   [tarot daily/draw/每日/運勢 (問題)](#daily)
        *   [tarot time/時間 (問題)](#time)
        *   [tarot cross/大十字 (問題)](#cross)
    *   [擲筊](#baobei)
        *   [cup (問題)](#cup)
*   [其他廢廢功能](#whatsthis)
    *   [隱藏關鍵字 (群組歡樂用)](#happyline)
*  [特別感謝](#acknowledgement)

<h1 id="trpg">TRPG</h1>

此處包含了 TRPG (Tabletop Role Playing Game) 會使用到的常見擲骰功能。

目前已支援 CoC 與 NC 骰子囉。（但是我沒玩過 DX 所以就不會寫...）


<h2 id="coc">CoC</h2>

CoC 為 Call of Cthulhu 的簡寫，相關說明請參考網路資源。（作者好懶）


<h3 id="coc6">coc6</h3>
    
CoC 六版擲骰語法，語法為，問題可為空白：
    
    ccb<=數字 (問題)
    
比方說：

    ccb<=50 San Check
    6 → 成功；SanCheck
    
    
<h3 id="coc7">coc7</h3>

CoC 七版擲骰語法，語法為，問題可為空白：

    cc<=數字 (問題)

比方說：

    cc<=50 SanCheck
    64 → 失敗；SanCheck
    
    
<h3 id="award">獎勵骰</h3>

語法為，問題同樣可為空白：

    cc(N)<=數字 (問題)
    
比方說：

    cc(2)<=50 偵查
    84、24、84 → 24 → 困難成功；偵查
    
    
<h3 id="roll">直接丟骰子</h3>

直接輸入 roll 會跳出使用說明，像是：

    roll
    
    總之你要擲骰前就先打roll，後面接像是2d6，1d6+3，2d6+1d3之類的就好。  
    要多筆輸出就是先空一格再打像是 *5 之類的。  
    請愛用小寫d，不支援大寫D

實際語法為，問題同樣可為空白：

    roll 1d20 (問題)
    roll 2d6+6 (問題)
    roll 3d6 *5 (問題) <-記得要空格
    
比方說：

    roll 1d20 傷害
    16；傷害
    
    roll 2d6+6 SIZ
    3+1+6=10；SIZ
    
    roll 3d6 *5 隨便丟
    8；11；6；8；15；隨便丟
    
    
<h3 id="createcoc6">快速產生 CoC 6th 角卡</h3>

語法為，問題同樣可為空白：

    roll 六版角卡 (要填的話、角色名稱)
    
比方說：

    roll 六版角卡 Steven
    
    Steven:
    STR: 6+2+3=11
    CON: 6+1+2=9
    POW: 5+4+3=12
    DEX: 3+2+4=9
    APP: 1+2+6=9
    SIZ: 5+1+6=12
    INT: 5+6+6=17
    EDU: 1+1+3=5
    
看起來廢廢的。
    

<h3 id="createcoc7">快速產生 CoC 7th 角卡</h3>

同樣地語法為，問題同樣可為空白：

    roll 七版角卡 (要填的話、角色名稱)
    
比方說：

    roll 七版角卡 Steven
    
    Steven:
    STR: 50
    CON: 65

<h3 id="ncdice">NC 擲骰介紹</h3>

此版本已提供 NC 常見的擲骰功能囉（格式則參考跑團平台──凍豆腐）。
如要擲骰的話請參照下列格式：

    (骰數)NC/NA (問題)
    
比方說：

    1NC
    7[7] → 成功
    
    4nc-2
    5[2,1,5,0] → 大失敗
    
    1NA+2 反物資
    6[6] → 成功 ; 反物資
    
已支援同時大小寫判斷。

<h2 id="tarot">塔羅牌</h2>

此程式提供塔羅牌的抽牌功能，如果直接輸入 tarot 的話會跳出說明：
    
    tarot
    
    每日運勢: 22 張，請用 tarot daily/每日 
    時間之流: 79 張，請用 tarot time/時間 (問題) 
    大十字: 79 張，請用 tarot cross/大十字 (問題) 
    儘管玩玩看
    
也可以抽每日運勢，會從 22 張大牌內抽一張出來：

    tarot daily (問題)
    
    命運之輪 ＋
    
如果後面不加關鍵字，單純問問題，則會從 79 張牌（嘿嘿）內抽一張出來：

    tarot 隨便抽
    
    權杖一 ＋
    

<h3 id="daily">每日運勢 / 每日一抽</h3>

每天來一張吧！相關語法為：

    tarot daily/draw/每日/運勢 (問題)
    
比方說這樣輸入都可以：

    tarot daily 
    魔術師 ＋
    
    tarot 每日 隨便問問題
    命運之輪 － ; 隨便問問題


<h3 id="time">牌陣：時間之流</h3>

這是塔羅一個很經典的牌陣：時間之流，可以在精簡的分析內看到過去、現在與未來的概觀。如果想要用此牌陣的話，相關語法為：

    tarot time/時間 (問題)
    
比方說：

    tarot time 還是隨便問
    
    還是隨便問: 
    過去: 惡魔 ＋
    現在: 錢幣八 ＋
    未來: 隱者 －


<h3 id="cross">牌陣：塞爾特大十字</h3>

這是塔羅一個非常實用的牌陣，此牌陣能夠清楚地從各個面相顯示出問題的核心，真的遇到問題就用這個吧，甚至不需要學其他牌陣（大誤），相關語法如下：

    tarot 大十字/cross (問題)
    
比方說：

    tarot cross 作者帥不帥
    
    作者帥不帥: 
    現況: 權杖皇后 ＋
    阻力: 寶劍四 －
    目標: 錢幣五 －
    基礎: 戰車 －
    過去: 聖杯皇后 －
    未來: 寶劍騎士 －
    自我: 權杖侍者 －
    環境: 惡魔 ＋
    希望: 寶劍二 ＋
    結論: 寶劍九 －
    
就是一個很經典的自己被表的例子（好樣的）。相關說明請參閱網路其他資源，本軟體暫時不提供解牌（推薦自己解）

<h2 id="baobei">擲筊</h2>

此杯杯機器人並提供了簡單的擲筊功能，設計含意如下：

* 使用 0 為陰數， 1 為陽數，用陰陽交替的方式來取杯。
* 為了呈現出擲筊的多樣性，因此多引入數字 2 當做輔助數。
* 程式會自動丟三次杯，如特定杯有超過一定次數則取杯，反之則沒杯。
* 此程式會出現聖杯、笑杯、蓋杯、沒杯、立杯等現象。
* 沒杯的可能性很高。


另外在這邊簡單定義一下各種杯的含意：

* 聖杯：好、贊成、OK。（不是聖杯戰爭那個。）
* 笑杯：答案自在心中不需要問、神明在笑、問的很接近了但又差一點點。
* 蓋杯：不好、反對、NO。
* 沒杯：不想回答、不置可否、問的問題太過偏遠、不切實際。
* 立杯：哈哈神祕現象哈哈，請自行解讀。


<h3 id="cup">來丟個杯吧</h3>

如果直接輸入以下語法，則會跳出說明：

    cup 說明
    
    常見的擲筊 
    請用 丟杯/擲筊/cup (問題) 
    會出現聖杯、笑杯、蓋杯、沒杯與立杯等結果
    
    
如果想要丟杯，則使用下列語法：

    cup (問題)
    
比方說：

    cup
    聖杯
    
    cup 週末洗衣服好不好
    沒杯 ; 週末洗衣服好不好
    
代表這種小問題不需要來問我 T_T
